# Devman Reviewer Bot
Бот для отслеживания статуса проверки работ на платформе Devman. При изменении статуса проверки отправляет уведомления в Telegram.

## Функциональность
Отслеживание статуса проверки работ на Devman

Автоматические уведомления в Telegram при проверке работы

Информация о результате проверки (успешно/нужны доработки)

Ссылки на проверенную работу


## Установка

### Установите зависимости:

```
pip install -r requirements.txt
```
### Настройка
1. Получение токенов
Telegram Bot Token:

Откройте Telegram и найдите @BotFather

Создайте нового бота командой /newbot

Сохраните полученный токен

Devman API Token:

Авторизуйтесь на dvmn.org

Перейдите в раздел API

Скопируйте ваш токен


Chat_ID:

Узнайте ваш chat_id через бота @userinfobot

Скопируйте полученный chat_id

2. Настройка переменных окружения
Создайте файл .env в корне проекта и добавьте:
```
TG_TOKEN=ваш_telegram_токен
DEVMAN_TOKEN=Token ваш_devman_токен
TELEGRAM_CHAT_ID=ваш_chat_id_если_известен
```


## Использование
Запустите бота:
```
python main.py
```
